# IMS Managed Services Executive Mini-Deck

Open `executive-mini-deck/index.html` in a browser to review the 3-slide responsive HTML deck. Each slide is designed for 16:9 presentation use.

## Present

Use the browser in full-screen mode, or print/export the deck to PDF for review.

## Export to PDF

1. Open the deck in Chrome or Edge.
2. Choose **Print**.
3. Set destination to **Save as PDF**.
4. Select landscape orientation.
5. Enable **Background graphics**.
6. Use no margins when available.

The CSS uses page breaks so each slide exports as a separate PDF page.
