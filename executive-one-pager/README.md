# IMS Managed Services Executive One-Pager

Open `executive-one-pager/index.html` in a browser to review the responsive briefing sheet.

## Export to PDF

1. Open the page in Chrome or Edge.
2. Choose **Print**.
3. Set destination to **Save as PDF**.
4. Use **Background graphics** so the IMS navy and gold panels export correctly.
5. Use default margins or reduce margins if a tighter one-page PDF is preferred.

The CSS includes print rules to preserve color and avoid awkward section breaks where possible.
